# 腊肠狗吃鱼油

一个有趣的贪吃蛇游戏，主角是可爱的腊肠狗！

## 游戏特点

- 🐕 可爱的腊肠狗造型
- 🐒 调皮的猴子会扔粑粑
- 💊 收集鱼油补充营养
- 📱 支持手机触屏操作

## 如何游玩

1. 使用键盘方向键或触屏按钮控制腊肠狗移动
2. 收集鱼油增加分数和长度
3. 每收集5个鱼油会清除场上的粑粑
4. 注意避开猴子扔的粑粑
5. 撞到墙壁或自己的身体会游戏结束
